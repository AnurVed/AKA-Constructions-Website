# AKA-Constructions-Website

A responsive corporate website built for **A.K.A. Constructions**, designed to improve client engagement and online visibility.

---

## 🚀 Project Overview
This project showcases a modern construction company website with:
- Clean, responsive design for desktop and mobile
- Easy navigation across services and projects
- Integrated communication tools for client outreach

---

## ✨ Features
- **Google Maps Integration** – for location visibility
- **WhatsApp Messaging** – instant client communication
- **Responsive Layout** – optimized for multiple devices
- **Project & Services Pages** – structured content for business offerings

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Next.js
- **Styling:** Tailwind CSS / PostCSS
- **Version Control:** Git & GitHub
